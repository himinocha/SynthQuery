SynthQuery Documentation

Explore the functionality and features of SynthQuery with our comprehensive command guide.

cre-db (Create)

cre-db is SynthQuery’s command for creating a database. Users are prompted to enter the name of the database they want to create. A message indicating success or error follows, depending on whether the database already exists.

cre-tb (Create)

cre-tb is SynthQuery’s command for creating a table. Users are prompted to enter the name of the database, the name of the table, and the format of the table (CSV or JSON). A success or error message appears based on the validity of the database and the existence of the table name.

del-db (Delete)

del-db is the command to delete a database. Users enter the name of the database they want to delete. A success or failure message follows, depending on the database's existence.

del-rows (Delete)

del-rows is the command to delete rows from a table based on specified conditions. Users enter the name of their database and table, followed by deletion conditions as a JSON string. Validity checks are performed on the database and table names.

ins-cval (Insert)

ins-cval is the command used to insert CSV values into a specified database and table. Users enter the names of their desired database and table, followed by values as a JSON string. A success or error message appears based on the validity of the entered information.

ins-jval (Insert)

ins-jval is the command used to insert JSON values. It follows the same steps as ins-cval, with the difference being that users are prompted to insert JSON string values within square brackets.

project-col (Projection)

project-col is the command used to select columns of a table. Users enter the names of their desired database and table, followed by the columns they want to select as a comma-separated list, or a blank input to select all columns.
